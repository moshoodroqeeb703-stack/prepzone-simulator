 # PrepZone Simulator

Nigeria's most complete exam simulator for WAEC, NECO & JAMB.

## Features
- Real past questions from 2001–2020
- WAEC, NECO & JAMB sections
- Science, Commercial & Arts categories
- Access code system — teachers buy codes, students use them
- Teacher dashboard with live student score tracking
- Paystack payment integration
- PWA — installable on any phone
- Dark mode, mobile-first design
